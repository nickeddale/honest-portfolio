Honest Portfolio Competitive Analysis

Honest Portfolio - Competitive Analysis Report

Date: December 2025

Purpose: Market validation, feature planning, and competitive positioning

Executive Summary

Honest Portfolio occupies a unique niche: a free PWA focused specifically on opportunity cost calculation for individual stock picks vs. benchmark alternatives (SPY, AAPL, META, GOOGL, NVDA, AMZN). While many portfolio trackers offer benchmark comparison as a secondary feature, none make it the central value proposition with per-purchase drill-down analysis.

Key Finding

No direct competitor offers the exact combination of:

  • Per-purchase opportunity cost analysis
  • Multiple individual stock benchmarks (not just indices)
  • Visual time-series showing "paths diverging"
  • Free, simple PWA with no account required

Detailed Competitor Analysis

Category 1: Portfolio Trackers with Benchmark Features

Empower (formerly Personal Capital) - empower.com

Type: Free web + mobile app

Benchmark Feature: "You Index" compares portfolio against S&P 500, DOW, bonds, foreign markets

Strengths:

  • Completely free
  • Connects to 17,000+ financial institutions
  • Investment Checkup tool with sector analysis
  • Efficient frontier visualization

Weaknesses vs Honest Portfolio:

  • Aggregate portfolio only (no per-purchase analysis)
  • Only compares to market indices, not individual stocks
  • Doesn't show "what if you bought X instead" for each purchase

Sharesight - sharesight.com

Type: Web + mobile, global focus

Price: Free (10 holdings), $7-49/month for paid

Benchmark Feature: Compare portfolio against 700K+ stocks, ETFs, funds

Strengths:

  • Tax reporting (CGT, dividends)
  • Global coverage (40+ exchanges)
  • True return calculation (TWRR)
  • Assumes equal investment in benchmark for fair comparison

Weaknesses vs Honest Portfolio:

  • Aggregate comparison only
  • Paid for meaningful use
  • Benchmark comparison is secondary feature

Delta by eToro - delta.app

Type: Mobile-first (iOS/Android)

Price: Free, PRO $99/year

Benchmark Feature: Compare portfolio beta and P/E to SPY

Strengths:

  • Best free mobile experience
  • Crypto + stocks + NFTs
  • Best Portfolio Tracker 2024 (Benzinga)

Weaknesses vs Honest Portfolio:

  • No per-purchase opportunity cost
  • Limited benchmark comparison
  • eToro branding/upsells

Category 2: Backtesting & Analysis Tools

Portfolio Visualizer - portfoliovisualizer.com

Type: Web-based analysis tool

Price: Free (limited), $39/month Pro

Benchmark Feature: Compare up to 3 portfolios against any benchmark

Strengths:

  • Monte Carlo simulation
  • Historical backtesting
  • Factor analysis
  • Efficient frontier

Weaknesses vs Honest Portfolio:

  • Manual data entry (no broker sync)
  • Complex interface for casual users
  • Not a tracker, more an analysis tool

Category 3: S&P 500 / "What If" Calculators

DQYDJ S&P Calculator

dqydj.com

Historical S&P 500 returns with dividend reinvestment. Accounts for taxes, fees, inflation.

Gap: Single benchmark only, no portfolio connection

Clark Howard Calculator

clark.com

Simple, beginner-friendly. Historical data back to 1975.

Gap: S&P 500 only, no portfolio connection

Of Dollars and Data

ofdollarsanddata.com

DCA calculator with data back to 1871.

Gap: Not a tracker, index only

Recommendations

Market Validation

  • Strong signal: No direct competitor focuses on per-purchase opportunity cost
  • Validation: Forum discussions show users asking "how do I compare my picks to index?"
  • Caution: Niche appeal; may be too "painful" for some users

Feature Planning

High Value:

  • Broker integration (Plaid)
  • Custom benchmark selection
  • Dividend tracking

Medium Value:

  • CSV import/export
  • Native mobile app
  • Social sharing

Positioning/Marketing

Tagline Ideas:

  • "The portfolio tracker that doesn't lie to you"
  • "Would you have been better off with SPY?"
  • "The regret calculator for investors"

Key Differentiators:

  • Only tracker focused on opportunity cost
  • Per-purchase analysis
  • Free, no signup required
